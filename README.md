# Voltmeter

DAC (MCP4725):

Vref = Vdd = 5V
Wejscie 12-bitowe.
Wyjscie w przedziale 0-Vref.

Dla wartości 1000 zwraca napięcie 1.22V.
 
1000/4096 = 1.22V/5V


ADC (MCP3424):

Vref = 2.048V

Dla napięcia 1.22V zwraca wartość 19480.
Wejscie w przedziale -Vref - Vref.
Wyjsce 16-bitowe.

19480/32767 = 1.22V/2.048V
